# Basic MarkdownUI Examples

## Buttons
__Primary Button|Click Me__

__Secondary Button|Another Action|secondary__

__Button|Disabled|disabled__

## Headers
__Large Header|Welcome__

__Medium Header|Section Title__

__Small Header|Subtitle__

## Messages
__Success Message|Everything worked perfectly!__

__Warning Message|Please be careful|warning__

__Error Message|Something went wrong|error__

## Segments
__Segment|This is a basic content segment.__

__Raised Segment|This segment is raised.|raised__

__Secondary Segment|This is secondary content.|secondary__
